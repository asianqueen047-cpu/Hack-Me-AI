# Hack-Me-AI

HACK ME AI is a responsive AI chatbot created with HTML, CSS, JavaScript, and PHP. It uses the Gemini API to generate smart and helpful conversational responses through a clean chat interface.

## Live Demo

https://hackmeai.xo.je/

## About

HACK ME AI is a lightweight web-based AI chat application. The frontend is built with HTML, CSS, and JavaScript, while the backend uses PHP to connect with the Gemini API and return AI-generated replies.

This project is designed to be simple, responsive, beginner-friendly, and suitable for PHP-supported web hosting.

## Features

* Responsive mobile-first chat interface
* Clean and modern AI chat UI
* Built with HTML, CSS, and JavaScript
* PHP backend for API handling
* Gemini API integration
* Typing indicator while the assistant is responding
* Multiple API key fallback support
* Basic error logging for debugging
* Lightweight and easy to deploy

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* AI API: Gemini API
* Hosting: PHP-supported web hosting

## How It Works

1. The user types a message in the chat box.
2. JavaScript sends the message to `api.php`.
3. `api.php` sends the request to the Gemini API.
4. Gemini API returns an AI-generated response.
5. The response is displayed inside the chat interface.

## Setup Instructions

1. Download or clone this repository.
2. Upload the project files to a PHP-supported hosting server.
3. Open `api.php`.
4. Add a valid Gemini API key.
5. Open the website in a browser.
6. Start chatting with HACK ME AI.
